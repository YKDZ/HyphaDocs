# 物品定义

无论是配置菜单图标还是商品，都需要撰写 `yml` 格式的物品配置。

插件中，物品被抽象为 `基本物品 + 装饰属性`。在配置中看到的 `base` 配置键即为基本物品配置，而其他诸如 `name`、`lore`、`item-flags` 等的属性都是装饰属性。

如果想要一个显示名称为 `<red>神奇泥土`、带有自定义 `lore`、且被 `附魔` 了 **击退 V** 的泥土，应该如何书写？

首先，明确基本物品是**原版物品**中的**泥土**：

```yaml
base: dirt
```

之后，按喜欢的顺序排列所需的装饰属性即可：

```yaml
base: dirt
name: "<red>神奇泥土"
lore:
  - "<gray>可以将目标击飞的神奇泥土"
enchantments:
  - knockback:5
```

这种配置方式在商品物品、商品图标、菜单图标等等地方都是**通用的**。
