# 命令与权限

## 商店

| 命令格式                                   | 权限                             | 作用                     | 备注                                                                        |
|--------------------------------------------|----------------------------------|--------------------------|-----------------------------------------------------------------------------|
| `hyphashop shop open <商店 ID> <玩家目标>` | `hyphashop.command.shop.open`    | 为玩家打开指定商店的菜单 | 打开菜单的目标玩家还需要拥有对应商店的 `hyphashop.shop.open.<商店 ID>` 权限 |
| `hyphashop shop restock <商店 ID>`         | `hyphashop.command.shop.restock` | 刷新指定商店内的商品     |                                                                             |

## 交易日志

| 命令格式                                     | 权限                             | 作用                                       | 备注                                                                                              |
|----------------------------------------------|----------------------------------|--------------------------------------------|---------------------------------------------------------------------------------------------------|
| `hyphashop history open <玩家名> [玩家目标]` | `hyphashop.command.history.open` | 为 `玩家目标` 打开 `玩家名` 的交易日志菜单 | 若玩家目标和玩家名不是一个人，则命令执行者需要额外拥有 `hyphashop.command.history.open.other` 权限 |

## 购物车

| 命令格式                                  | 权限                          | 作用                                 | 备注 |
|-------------------------------------------|-------------------------------|--------------------------------------|------|
| `hyphashop cart open <玩家名> [玩家目标]` | `hyphashop.command.cart.open` | 为 `玩家目标` 打开 `玩家名` 的购物车 |      |

:::info
插件使用 Paper Brigadier Command API 注册命令，所有命令中的 `玩家目标` 参数实际上都接受一个类似原版的目标选择器参数并从选择到的目标中筛选出第一个玩家。
:::

## 其他菜单

| 命令格式                                  | 权限                         | 作用                           | 备注 |
|-------------------------------------------|------------------------------|--------------------------------|------|
| `hyphashop gui open <菜单 ID> [玩家目标]` | `hyphashop.command.gui.open` | 为 `玩家目标` 打开 `<菜单 ID>` |      |

菜单的查看者还额外需要权限 `hyphashop.gui.open.[菜单 ID]`。
