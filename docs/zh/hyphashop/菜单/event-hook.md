# 事件钩子

| 名称           | 配置键             | 解释                           |
| -------------- | ------------------ | ------------------------------ |
| 菜单开启时     | `on-open`          | 菜单开启时触发一次             |
| 菜单关闭时     | `on-close`         | 菜单关闭时触发一次             |
| 点击菜单外部时 | `on-outside-click` | 玩家点击菜单外部区域时触发一次 |

:::tip
这些钩子也同样适用于商店菜单、购物车菜单等等特殊菜单。
:::
