# 数组

使用 `[]` 语法来声明数组字面量。

数组索引从 `0` 开始。

以下是常见的用法：

```js
const a = 5
const arr = [1, a, 3, 2 + 2, [1, 2]]

// 以下表达式的值都为 true
arr[1] == 5
arr[4].sum() == 3
```

## 切片

数组可以被切片访问，语法与 `Python` 相同：

```js
const arr = [1, 2, 3, 4]

// 以下表达式的值都为 true
arr[0:4].sum() == 10
arr[:2].sum() == 3
arr[-3:-2].sum() == 2
arr[::-1][0] == 4
arr[1:100:2].sum() == 6
```
