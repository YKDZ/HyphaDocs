# 事件钩子

普通图标的 `actions` 事件钩子即为玩家的鼠标、键盘操作。可以监听的操作如下：

|名称|配置键|解释|
| -------------| ------| -------------------------------------------------------------------------|
|左键点击|`left`|鼠标左键点击图标|
|Shift + 左键|`shift-left`|按住 Shift 并用鼠标左键点击图标|
|右键点击|`right`|鼠标右键点击图标|
|Shift + 右键|`shift-right`|按住 Shift 并用鼠标右键点击图标|
|点击左侧边框|`window-border-left`|点击菜单灰色边框的左侧部分（未观察出与 `left` 的区别，此处仅翻译 API 中的注释）|
|点击右侧边框|`window-border-right`|点击菜单灰色边框的右侧部分（未观察出与 `right` 的区别，此处仅翻译 API 中的注释）|
|鼠标中键点击|`middle`|鼠标中键点击图标|
|点击数字键|`number-key`|鼠标移到图标上，并按下 `1-9` 任意一个数字键|
|左键双击|`double-click`|快速左键点击图标两次|
|点击丢弃键|`drop`|鼠标移到图标上，并按下丢弃键（默认为 `Q`）|
|Ctrl + 丢弃键|`control-drop`|鼠标移到图标上、按住 Ctrl，并按下丢弃键（默认为 `Q`）|
|切换到副手|`swap-offhand`|鼠标移到图标上，并按下交换到副手键（默认为 `F`）|
|创造模式操作|`creative`|创造模式物品栏开启时的任意操作（在本插件中应没有实际意义）|
|未知操作|`unknown`|所有未知的操作（在本插件中应没有实际意义）|
